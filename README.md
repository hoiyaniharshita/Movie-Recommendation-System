# Movie Recommendation System

This project implements a movie recommendation system using collaborative filtering techniques.

The system analyzes user ratings and suggests movies based on user preferences.

## Technologies Used

- Python
- Apache Spark
- Spark MLlib
- Pandas

## Dataset

MovieLens dataset containing more than 100,000 movie ratings.

## Features

- Collaborative filtering using ALS algorithm
- Large scale data processing using Apache Spark
- Parameter tuning using RMSE evaluation
- Personalized movie recommendations

## Workflow

1. Load and preprocess movie ratings dataset
2. Train collaborative filtering model using ALS
3. Tune model parameters
4. Evaluate model performance using RMSE
5. Generate movie recommendations for users
